# 月日（つきひ）漢方診断サービス

## サービス概要
働く女性（30〜40代以降）向けの体質診断＋最適漢方薬推薦サービス。
簡単な質問に答えるだけで、自分の証（体質傾向）がわかり、適した漢方薬を提案される。

## 特徴
- 気血水＋寒熱・虚実・陰陽まで網羅
- ストレス・冷え・乾燥・婦人科症状にも対応
- 複合証判定（例：気虚＋瘀血）
- 禁忌情報を考慮した安全な推薦
- 診断結果のレーダーチャート可視化
- 診断結果のPDF保存対応

## 使用技術
- React + TypeScript
- Vite
- Tailwind CSS
- Recharts（レーダーチャート）
- html2pdf.js（PDF生成）
